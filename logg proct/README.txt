aplicacion web de facturacion

Este es un proyecto de aplicacion web de facturacion lo que garantiza la mejor interfaz y intuitiva a la hora de hacer una factura.

Paso de Ejecución del Proyecto:

    1. Instalación de Dependencias: Asegúrate de tener instalado Python
     en tu sistema. usa el  (-pip install -r requirements.txt) para descargar las bibliotecas

    2. Configuración de la Base de Datos: Asegúrate de tener un servidor de base de datos
     MySQL disponible. En el archivo app.py, se utiliza la librería flask_mysqldb para la conexión
     a la base de datos MySQL. Asegúrate de actualizar las configuraciones de conexión con tu 
     base de datos en la sección correspondiente del archivo config.txt.

    3. asegurate de crear una base de datos y 
    llamarla login y ve a (import o importar) 
    y importa la base de datos login    

    3. Ejecución del Server Una vez configurada
     la base de datos, puedes ejecutar el servidor
     Flask con el siguiente comando:

    -python app.py


    4. Acceso a la Aplicación: Abre tu navegador web y visita la dirección http://localhost:5000
